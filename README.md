# pragati14 (Description)
A lightweight, local-first polling application built with Vanilla JavaScript, CSS3, and HTML5 that uses Web Storage for real-time, serverless voting and user authentication.

#Project title - 📊 SNAPPOLL (Polling App)
SNAPPOLL is a lightweight, client-side polling application designed for quick decision-making and real-time feedback. Built with a "local-first" philosophy, it allows users to create polls, cast votes, and share results without the need for a complex backend.

*Features:-
 1) User Authentication
 2)Dynamic Poll Creation
 3)Real-time Expiry
 4)Visual Results
 5)Anti-Spam Protection 
 6)One-Click Sharing
 7)Cross-Tab Sync

*Tech Stack:-
  1)HTML5
  2)CSS3
  3)JavaScript (Vanilla)
  4)Web Storage API

*Getting Started:-
         Since SNAPPOLL is a pure frontend application, there are no dependencies which need to be installed.
     1)Clone or Download: Save the index.html, style.css, and script.js files into the same folder.
     2)Run: Open index.html in any modern web browser.
     3)Deploy: This app can be hosted for free on platforms like GitHub Pages, Netlify, or Vercel by simply uploading these              three files.
